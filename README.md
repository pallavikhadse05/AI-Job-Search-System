# AI-Job-Search-System
Built an agentic workflow that runs daily – fetching LinkedIn jobs via Apify, generating a tailored resume using Gemini AI, compiling it as a LaTeX PDF, and delivering it by email automatically.
I recently built an automated job search system using n8n, Railway, Apify, Supabase, Google Console, Google Drive, Gmail, Google Gemini AI, LaTeX.

The system runs every morning — fetches relevant LinkedIn jobs, customizes my resume for each role using AI, generates a PDF, and sends me a daily summary. Automatically.

Building it meant solving real problems — OAuth failures, API errors, workflow debugging, and JavaScript issues. No shortcuts.
